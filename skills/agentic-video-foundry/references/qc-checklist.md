# 短视频交付质检清单

## 内容与节奏

- [ ] 唯一传播结论能用一句话说明，CTA 明确。
- [ ] 前 1–3 秒有钩子，关键结论没有被清单式口播淹没。
- [ ] 信息落地后有停顿，运动高潮有限且有铺垫。
- [ ] 最终时长来自真实旁白和时间轴，不是机械加速所得。

## 画面

- [ ] 代表帧已真实渲染并人工检查，不只看代码或 Studio。
- [ ] 每帧只有一个 Hero；字幕、工具名、数字和 CTA 无截断、错字或遮挡。
- [ ] 顶部、左右、底部和右侧操作区符合目标平台安全区。
- [ ] Hook、转场前后、字幕边界、最强动效和 CTA 的精确帧均无闪烁、残影或层级污染。
- [ ] 无 CSS 运行时动画、计时器、时间依赖或无种子随机造成的不可复现行为。
- [ ] 产品/工具片包含真实 UI、操作、源码或输出证据；动效没有替代本应出现的 Demo。

## 声音

- [ ] 声线符合角色、语言和情绪，专有名词发音正确，无机械播音腔或过度叫卖。
- [ ] 背景音乐在手机外放上能清楚感知，仍不遮盖每一句旁白。
- [ ] 音效有叙事作用，无过密、拖尾或突兀峰值。
- [ ] 完整成片已测综合响度和真峰值；最终编码无削波、爆音、断裂或异常静音。
- [ ] 声线、BGM、SFX、增益或时间改变后，已重新渲染并重测，而非沿用旧数值。

## 技术与安全

- [ ] 分辨率、画幅、帧率、编解码器、音频采样率和时长符合交付规格。
- [ ] API Key 不在源码、manifest、日志、命令记录或成片元数据中。
- [ ] 付费资产存在哈希记录，未变化的素材可复用；使用的供应商和模型有证据。
- [ ] 音频供应商切换后已重新生成或校准时间轴、字幕、混音与母带；未沿用旧供应商的百分比和测量值。
- [ ] 字幕时间来自对齐数据或经人工校准，中英混排品牌短语未被拆断。

## 交付

- [ ] MP4、封面、字幕、最终脚本、发布文案、可编辑源工程和复现命令齐全。
- [ ] 新版成片通过验收前没有覆盖唯一可回退的旧版。
- [ ] 已完整观看并收听最终文件，而不是中间预览文件。
- [ ] 已记录本轮新发现；只有重复验证的模式才提升为长期规则或 skill。
